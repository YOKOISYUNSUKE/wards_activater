# \# 病床管理アプリ

# 

# 病院の病床管理・退院調整を支援するWebアプリケーション

# 

# \## 機能

# \- 病棟ごとの患者管理

# \- DPCコード自動補完

# \- 退院調整アルゴリズム

# \- 看護必要度計算

# \- 稼働率・平均在院日数のKPI表示

# 

# \## 使い方

# 1\. アプリにアクセス: https://\[your-username].github.io/\[repo-name]/

# 2\. IDとパスワードを登録してログイン

# 3\. 病棟を追加して患者情報を入力

# 

# \## 技術スタック

# \- Pure JavaScript (ES6+)

# \- IndexedDB (ローカルストレージ)

# \- GitHub Pages (静的ホスティング)

# 

# \## ローカル開発

# ```bash

# \# HTTPサーバーで起動

# python -m http.server 8000

# \# または

# npx http-server

# ```

# 

# \## 注意事項

# \- データはブラウザのIndexedDBに保存されます

# \- 端末を変えるとデータは共有されません
